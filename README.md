# 🎥 YouTube Playlist Title Scraper

This Python script extracts all video titles from any public YouTube playlist and saves them into a Word document (`.docx`).

## 📌 Features
- Fetches all video titles from a YouTube playlist using the YouTube Data API v3
- Saves the titles in a nicely formatted Word document
- Easy to modify for CSV, text, or PDF output

## 🛠️ Requirements

Install dependencies with:

```bash
pip install -r requirements.txt



# 🔑 Setup
# 1. Get your YouTube Data API v3 key
# 2. Create a config.py file in the root directory with:

# 3. Run the script: python youtube.py

# 📂 Output
